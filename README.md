# 🛒 Supermarket Intelligence & POS System (DAIRIP)

A distributed retail platform featuring a barcode-scanning client interface and a robust administrative control panel.

## 🌟 Key Features

### 📱 Client-Side Application
* **Barcode Scanning:** Instant product identification using device camera/scanner.
* **Real-time Search:** Integrated with a central database to fetch product details (price, stock, description).
* **Smart Cart:** Add/remove items with automatic subtotal calculation.
* **M-Pesa Integration:** Secure checkout powered by the **Safaricom Daraja API** (STK Push).

### 🔐 Admin Panel
* **Inventory Management:** Add, update, or delete products and track stock levels.
* **Transaction Logs:** View real-time sales data and M-Pesa payment confirmations.
* **Analytics:** Basic insights into top-selling products.
